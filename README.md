# PS-17 Plastic to Product Marketplace
A full-stack web platform that enables users to buy and sell products made of plastic, promoting sustainability and circular economy practices.  
Sellers can list eco-friendly plastic-based products, while buyers can browse, search, and purchase items easily.  
The system includes a dedicated Admin panel to manage users, product listings, and platform activities.  
Built with a responsive frontend and a Python-based backend, ensuring secure data handling and smooth transactions.  
Designed for cross-device compatibility with future scope for online payments, analytics, and cloud deployment.
